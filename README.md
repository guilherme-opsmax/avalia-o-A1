### 1️⃣ Verificação de Idade para Eventos
Descrição

Programa que solicita nome, idade e o tipo de evento (Livre, 12+, 16+ ou 18+).
Com base nesses dados, o sistema informa se a pessoa pode ou não entrar no evento escolhido.

Como executar
python verificar_idade.py

Exemplo de entrada
Digite seu nome: Pedro
Digite sua idade: 15
Escolha o tipo do evento (1-4): 3

Exemplo de saída
❌ Pedro, você NÃO pode entrar. Idade mínima: 16 anos.

### 3️⃣ Cadastro de Alunos
Descrição

O programa recebe nomes digitados pelo usuário e os armazena em uma lista.
A entrada continua até que o usuário digite "sair".
Ao final, todos os alunos cadastrados são exibidos.

Como executar
python cadastro_alunos.py

Exemplo de entrada
Digite o nome do aluno: João
Digite o nome do aluno: Ana
Digite o nome do aluno: Marcos
Digite o nome do aluno: sair

Exemplo de saída
=== Lista de Alunos Cadastrados ===
- João
- Ana
- Marcos

### 4️⃣ Sistema de Cadastro de Produtos
Descrição

Sistema simples para cadastrar produtos, onde cada produto contém:

Nome

Preço

Os produtos são armazenados em dicionários dentro de uma lista.
A inserção encerra quando o usuário digita "sair" como nome.

Como executar
python cadastro_produtos.py

Exemplo de entrada
Nome do produto: Arroz
Preço do produto: 12.50
Nome do produto: Feijão
Preço do produto: 8.99
Nome do produto: sair

Exemplo de saída
=== Lista de Produtos Cadastrados ===
- Arroz | R$ 12.50
- Feijão | R$ 8.99
